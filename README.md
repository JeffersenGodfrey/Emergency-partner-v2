# Emergency Partner

AI-powered offline emergency resource finder. React + Node.js.

## Setup

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

App runs on `http://localhost:3000`

## Development

**Lint/Format:**

```bash
npm run lint
npm run format
```

**Build:**

```bash
npm run build
```

## Stack

- React 18 + Vite
- Node.js + Express
- ESLint + Prettier
- GitHub Actions CI/CD
